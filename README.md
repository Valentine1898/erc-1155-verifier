```sh
$ pnpm install
```

### Compile

Compile the smart contracts with Hardhat:

```sh
$ pnpm compile
```


### Test

Run the tests with Hardhat:

```sh
$ pnpm test
```


### Deploy

Deploy the contracts to Mumbai :

```sh
$ pnpm deploy:contracts --network polygon-mumbai
```

### Tasks

Set requests:

```sh
pnpm task:setRequests --network polygon-mumbai
```
